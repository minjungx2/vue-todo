# _**vue project**_
**Vue.js로 만드는 기록도구**



#### Project setup
```
npm install
```

#### Compiles and hot-reloads for development
```
npm run serve
```

#### Compiles and minifies for production
```
npm run build
```

#### router install
```
npm install vue-router@next
```

#### Bootstrap install
```
npm install --save bootstrap
```
