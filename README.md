# _**vue project**_
**Vue.js로 만드는 기록도구**



#### Project setup
```
npm install
```

#### Compiles and hot-reloads for development
```
npm run serve
```

#### Compiles and minifies for production
```
npm run build
```

#### router install
```
npm install vue-router@next
```

#### Element ui install
```
npm install element-plus --save
npm install @element-plus/icons-vue
```
